1 在github上新建版本库 New repository

命名为app4
2 在本地新建项目app4
3 打开git bash 切换到app4根目录
4 git clone 版本库的git地址，在版本库主页可以找到，以.git结尾
git clone http://github.com/liumengni/app4.git
5 cd app4
6 cp -r .git ../ 将app4中的所有文件都复制到项目的根目录下，包括.git .gitignore license readme.md 然后删除app4文件夹
7 将项目的文件提交到github
git add .
git commit -m "first commit"
git push origin master 
输入用户名和密码
